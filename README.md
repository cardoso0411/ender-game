## 📝 Descrição

O **Ender Game** é um projeto de loja virtual de jogos digitais, desenvolvido como parte de um portfólio de estudos em desenvolvimento web. O site apresenta um catálogo diversificado de jogos para várias plataformas, com categorias que incluem ação, aventura, RPG, terror, simulação e muito mais.

## 🚀 Funcionalidades

- **Catálogo de Jogos**: Navegue por uma vasta coleção de títulos organizados por categorias
- **Design Responsivo**: Interface adaptável para diferentes tamanhos de tela
- **Interface Intuitiva**: Navegação simples e amigável
- **Categorias Diversificadas**: Jogos de ação, aventura, RPG, terror, simulação, lançamentos e mais

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura das páginas
- **CSS3**: Estilização e design responsivo
- **Bootstrap 4.5.2**: Framework CSS para componentes responsivos
- **JavaScript**: Funcionalidades dinâmicas (futuro)

## 📸 Pré-visualização

O projeto apresenta um tema escuro com acentos em amarelo e laranja, remetendo à cultura gamer. O design inclui:

- Header com navegação
- Cards de jogos com imagens
- Layout responsivo com Flexbox/Grid
- Footer com informações adicionais

## 🔧 Personalização

Para personalizar o projeto:

1. **Cores**: Edite o arquivo `style.css` para alterar o tema visual
2. **Imagens**: Adicione suas imagens na pasta `img/`
3. **Jogos**: Edite os arquivos HTML correspondentes para adicionar ou remover títulos
